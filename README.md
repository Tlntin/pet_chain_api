# pet_chain_api
度小满金融（原百度金融）区块链项目——莱茨狗，API python SDK文档

## 目前已实现的功能

1. 登录莱茨狗
2. 获取最新公告
3. 获取狗窝、微积分、元气值
4. 获取任务完成情况
5. 对狗狗市场进行查询（按价格，时间，稀有度排序）
6. 按id查询单只狗狗
7. 微积分一键收取
8. 购买宠物（~~积分不足，只有30多，暂未测试~~）
9. 获取验证码（~~自动打码~~）
